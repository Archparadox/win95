# win-port

A Windows 95-inspired portfolio site built with Next.js, React, and TypeScript.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run start
```

## Routes

- `/` - desktop-style portfolio experience
- `/about` - standalone about page
- `/work` - standalone work history page
- `/projects` - standalone featured project page
- `/components` - UI component showcase

## Notes

- Portfolio content lives in `data/portfolio.ts`
- Shared content renderers live in `components/portfolio`
- Windows 95-style UI primitives live in `components/ui`
- Planning-note workflow for agents lives in `AGENTS.md` and `docs/niarbdnoces/README.md`
